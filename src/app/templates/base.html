TMR4I Guideline Explorer

This web application allows you to explore several example clinical guidelines represented according to the TMR4I model developed by Veruska Zamborlini. A paper about this work has been submitted to the Semantic Web Journal: "Detecting Interactions among Medical Recommendations in Clinical Guidelines enhanced by the Semantic Web - A case study on multimorbidity", Veruska Zamborlini, Rinke Hoekstra, Marcos da Silveira, Cedric Pruski, Annette ten Teije and Frank van Harmelen.
